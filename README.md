This project is based on real data scraped from https://insideairbnb.com/get-the-data/ and https://www.statistik.at/fileadmin/pages/222/SpecialEvaluation.ods for the price of the Square Meters in the different districts. The dashboard aims to answer what district and property size would provide the best Return Of Investment (ROI). Categories of data that would result in outliers and a skewed view on the data was removed. This consists of things such as properties being rented out for a minimum of 30 days, properties with very high annual return or properties that were rented out less than 30 days in a year.
