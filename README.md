# PowerBI Vienna AirBnb ROI Analysis

The dashboard answers what district and property size would provide the best Return Of Investment (ROI). 

## Data Sources:
- [Inside AirBnb] (https://insideairbnb.com/get-the-data/)

- [Statistik Austria] (https://www.statistik.at/fileadmin/pages/222/SpecialEvaluation.ods)

## Data Cleaning
Categories of data that would result in outliers and a skewed view on the data was removed. This consists of things such as:
- Properties being rented out for a minimum of 30 days
- Properties with very high annual return or properties that were rented out less than 30 days in a year.

 ## Key Findings
2 Bedroom in Döbling is suitable if high risk investments are of interest. This as it has a higher ROI but it is based on only 15 listings.

1 Bedroom in Brigittenau is suggested for a safer investment as it is based on 152 listings.
